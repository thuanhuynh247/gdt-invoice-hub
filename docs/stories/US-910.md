# US-910: VBA Excel Login Upgrade - Webapp Smart Invoice API Integration

## Status

implemented

## Lane

high_risk

## Product Contract

Update VBA code in TaiHoaDonDienTu_v6.2.xlsm to use the local Webapp XML Smart Invoice API (`/API/login`) for login with auto-captcha solving. The webapp handles CAPTCHA resolution internally using vector signature matching + ddddocr OCR fallback.

---

## Architecture Decision

### Problem
The VBA Excel macro (`TaiHoaDonDienTu_v6.2.xlsm`) currently implements its own SVG CAPTCHA detection using `modDetectCaptcha.detectSVGCaptcha()`. This approach has limitations:
1. The VBA-side signature library is static and cannot learn new character variants
2. When detection fails, the user must manually type the captcha
3. No OCR fallback exists in VBA

### Solution
Route authentication through the **Webapp XML Smart Invoice API** (`/API/login`) which provides:
- **Two-phase captcha flow**: Phase 1 returns SVG + key, Phase 2 accepts solved captcha
- **Auto-solving**: The webapp uses vector signatures (matching VBA's approach) + ddddocr OCR fallback
- **Self-learning**: The webapp commits new vector signatures after successful logins
- **Retry logic**: Automatic retry with fresh captcha on failure

### Fallback
If the webapp is unavailable (not running), the VBA automatically falls back to the original direct GDT login flow with local SVG detection.

## Login Flow Diagram

```
VBA Excel (TaiHoaDonDienTu_v6.2.xlsm)
    │
    ├── [Smart Mode] ──► Webapp XML (localhost:5000)
    │       │                  │
    │       │   POST /API/login (phase 1: get captcha)
    │       │                  ├── Fetch SVG from GDT
    │       │                  ├── Auto-solve via vector sigs + OCR
    │       │                  └── Return {content, key}
    │       │
    │       │   POST /API/login (phase 2: authenticate)
    │       │                  ├── Submit to GDT with solved captcha
    │       │                  └── Return JWT token (text/plain)
    │       │
    │       └── Store JWT token → bearer
    │
    └── [Legacy Mode] ──► GDT Portal (hoadondientu.gdt.gov.vn:30000)
            │
            ├── GET /captcha → SVG
            ├── detectSVGCaptcha() → local solve
            └── POST /security-taxpayer/authenticate → JWT token
```
