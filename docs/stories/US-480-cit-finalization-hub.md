# Spec: US-480 — Interactive Form 03/TNDN Builder & CIT Finalization Hub

## Status
completed

## Lane
normal

## Product Contract

The system provides an **Annual CIT Finalization Dashboard** at `/v36-cit-finalization` designed with glassmorphic styling, containing the standard Form 03/TNDN fields and Phụ lục 03-1A details, permitting manual adjustment of revenue, expenses, and non-deductible items.

## Acceptance Criteria

- [x] Web view `/v36-cit-finalization` is accessible, with responsive layout and theme integration.
- [x] Displays input fields for Revenue, Operating Cost, and Non-Deductible Expenses.
- [x] Dynamically calculates Net Taxable Profit and estimated CIT liability.
- [x] Displays a business result card layout simulating GDT Phụ lục 03-1A.
