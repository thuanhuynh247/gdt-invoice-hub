# US-691: RF Exemption Auditor

## Story
**As a** tax compliance auditor,
**I want** to verify registration fee exemptions for qualifying assets,
**So that** exempt entities are correctly identified and zero-rated.

## Acceptance Criteria
- Agricultural/forestry land allocated by State → 100% exempt.
- Foreign diplomatic mission assets → 100% exempt.
- Revolutionary merit family social-policy housing → 100% exempt.
- Agricultural land transferred within the same household → 100% exempt.
- Exemption reason and audit note persisted in tenant database.
