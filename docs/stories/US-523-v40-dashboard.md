# Story US-523: Interactive FCT & Related Party Compliance Dashboard UI

## Title
Interactive FCT & Related Party Compliance Dashboard UI

## Description
To make compliance accessible, this story designs a premium dashboard command center for CFOs to manage FCT liability, Related Party EBITDA limitations, and digital signature audit trails.
The page is served at `/v40-compliance-dashboard` and features tabs/sections for:
- FCT Calculator & suggest Form 01/NTNN.
- Related Party EBITDA cap simulation slider.
- XML file upload & certificate verification checker.

## Design Guidelines
- Vibrant glassmorphic card elements with dark/light mode harmonized theme.
- Animated micro-interactions on calculators and sliders.
- Embed interactive compliance summaries.

## Verification
- View dashboard rendering, API checks, and simulation capabilities.
