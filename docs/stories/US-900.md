## Product Contract

The codebase is audited using the Ponytail Over-Engineering Auditor. Remediation will prioritize standard YAGNI rules:
1. **Reduce code nesting and complexity** where indentation exceeds 16 spaces/4 tabs, breaking logic into helper functions.
2. **Optimize file length** by splitting oversized modules into modular components.
3. **Refactor library dependencies** to minimize external dependency bloat (e.g., standardizing on Python's built-in libraries like `urllib.request` or `pathlib` where appropriate and feasible).
4. **Ensure full regression test coverage** to guarantee no functional modifications occur during refactoring.
